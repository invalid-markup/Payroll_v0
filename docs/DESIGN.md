# Design System for PayEasy+HR
## Typography
- Font Family: Inter, sans-serif
- Base Size: 14px (text-sm) for data density, 16px (text-base) for general text.
- Currency: TZS with thousands separators and 0 decimal places.

## Color Palette
Primary Action: #2563EB (Tailwind blue-600)
Secondary Action: #E5E7EB (Tailwind gray-200)
Success: #DCFCE7 (Tailwind green-100)
Warning: #FEF3C7 (Tailwind amber-100)
Danger: #FEE2E2 (Tailwind red-100)
Neutral: #F3F4F6 (Tailwind gray-100)
Info: #DBEAFE (Tailwind blue-100)
Filed / Complete: #D1FAE5 (Tailwind emerald-100)
Amended: #F3E8FF (Tailwind purple-100)

## Appearance
- Light Mode Default
- Clean, minimal, high data density.
- Rounded corners for cards: standard Tailwind rounded-lg.
